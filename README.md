# QuestIQ 프론트엔드

AI 기반 학생 질문 역량 진단 플랫폼 — React + TypeScript

## 빠른 시작

```bash
npm install
echo "REACT_APP_API_URL=http://localhost:3000" > .env.local
npm start
```

## Railway/Vercel 배포 환경변수

| 변수 | 설명 | 예시 |
|------|------|------|
| `REACT_APP_API_URL` | 백엔드 API URL | `https://api.railway.app` |
